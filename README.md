## 快速入门

### 开发

```bash
$ npm i               # 推荐用yarn
$ npm start           # 开发,CmdOrCtrl+Shift+V为截图快捷键
$ npm run build       # 打包
```

目录说明:

```
├─ dist           # 打包目录
├─ src
│  └─ main.js    #入口文件
│  └─ index.html     # 渲染进程的页面
```

### 安装

- dist目录下安装dmg
